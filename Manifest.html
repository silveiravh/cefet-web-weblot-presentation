Manifest

Web Manifest API

É um arquivo JSON simples que informa ao navegador sobre seu aplicativo da Web e como ele deve se comportar quando "instalado" no dispositivo móvel ou na área de trabalho do usuário.

Propriedades

  • short_name and/or name
  • icons
  • start_url
  • background_color
  • display
  • orientation

Exemplo de manifest

Telas Iniciais

Quando seu aplicativo é lançado pela primeira vez, pode demorar um pouco para o navegador acelerar e o conteúdo inicial começar a renderizar. o Chrome mostrará uma tela inicial, até a primeira tela

O Chrome criará automaticamente a tela inicial a partir das propriedades do manifesto, incluindo:

  • nome, cor de fundo, ícones

Testar o Manifest

Para verificar manualmente se o seu manifesto está configurado corretamente, você pode usar a guia "Manifest" no painel "Aplicativo" do Chrome DevTools.
